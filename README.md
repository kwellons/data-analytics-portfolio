# Data Analytics Portfolio
This portfolio highlights end-to-end data analytics projects that demonstrate my ability to clean, analyze, and interpret data using SQL, Python, and business intelligence tools. Each project focuses on solving real business problems, uncovering actionable insights, and translating data into clear recommendations for decision-making.

## Projects
### 1. Profit and Discount Analysis (SQL)
- Performed end-to-end SQL analysis of retail sales data to evaluate profitability drivers and pricing strategy effectiveness  
- Identified a clear profitability threshold where discounts above ~20% consistently lead to margin erosion and negative profit outcomes  
- Demonstrated that aggressive discounting does not reliably increase demand, challenging common assumptions around promotional strategy  
- Provided data-driven recommendations to improve pricing discipline and strengthen overall margin performance

### 2. HR Attrition Analysis (SQL & Power BI)
-  Analyzed workforce attrition patterns using SQL and Power BI to identify high-risk employee segments and key turnover drivers  
- Built an interactive dashboard highlighting attrition trends across tenure, job level, overtime status, and compensation levels  
- Discovered that employees with low tenure, overtime requirements, and lower income experienced attrition rates exceeding 57%, indicating a high-risk workforce segment  
- Delivered actionable insights and retention strategies using KPI analysis and diagnostic workforce analytics 

[View Full Project ->](HR Attrition Analysis/README.md)

### 3. Customer Behavior & RFM Segmentation Analysis (Python)
- Developed a customer segmentation model using RFM (Recency, Frequency, Monetary) analysis to evaluate purchasing behavior in an e-commerce dataset  
- Cleaned and transformed transactional data using Python (pandas), including feature engineering at the customer level  
- Identified key customer segments such as Champions, Loyal Customers, and At-Risk users based on behavioral patterns  
- Found that a small segment of high-value customers drives a disproportionate share of revenue, while a significant at-risk segment presents churn concerns
